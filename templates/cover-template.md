# Template de Prompt — Capa do Capítulo

## Objetivo

Gerar um prompt padronizado para criação das capas visuais dos capítulos da série.

## Estrutura do prompt

```
Capa para artigo técnico sobre [tema do capítulo], estilo [minimalista/flat design/tech],
cores [paleta de cores da marca], contendo elemento visual relacionado a [ícone ou metáfora do conceito],
texto "[Capítulo X — Título]" em destaque, proporção [16:9 | 1:1], sem elementos genéricos de IA.
```

## Variáveis a preencher

- **Tema:** [conceito central do capítulo]
- **Estilo visual:** [linha visual definida para a série]
- **Paleta de cores:** [cores oficiais do projeto]
- **Elemento/ícone:** [metáfora visual do conceito]
- **Título exibido:** [Capítulo X — Título]
- **Proporção:** [dimensão de uso: blog, LinkedIn, etc.]

## Onde usar a capa

- [ ] Blog
- [ ] LinkedIn
- [ ] README / SERIES.md
- [ ] GitHub Release
