# ⭐ Por que este projeto é diferente?

- ✅ **Projeto Open Source** — todo o conteúdo está disponível para leitura, estudo e colaboração.
- ✅ **Evolução capítulo por capítulo** — cada versão entrega um conceito novo, sem sobrecarregar quem aprende.
- ✅ **Código comentado** — exemplos pequenos com explicações diretas em cada passo.
- ✅ **Engenharia de Automação** — foco em qualidade, não apenas em fazer o teste passar.
- ✅ **Boas práticas de mercado** — exemplos que seguem padrões reais de projetos profissionais.
- ✅ **Exemplos reutilizáveis** — cada arquivo ensina um conceito isolado e aplicável em outras bases.
- ✅ **Documentação completa** — referência clara para estudante e para quem consulta depois.

# 🚀 Primeiros passos

```bash
git clone https://github.com/mayconmalmeida/playwright-typescript-guide.git
cd playwright-typescript-guide
npm install
npx playwright install --with-deps
```

Para verificar os testes e os tipos antes de contribuir:

```bash
npm test
npx tsc --noEmit
```
